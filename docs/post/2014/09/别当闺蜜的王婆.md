连岳：

您好！

我曾经给您发过邮件，但是您没回，我猜想我上次问的问题您可能觉得我无病呻吟，或者您觉得话题太俗套，不能激起您回信的欲望吧。

这次我同样问一个很俗套的问题，很可能您还是不会回信，不过没关系，就当您是个树洞吧。

我有个很要好的闺蜜，好得不能再好，她对我很好，在我最困难的时候陪我一起度过，我们彼此很依赖对方。

但是我们的性格，或者说人生观、价值观却有很大的差异，周围有人表露过不能相信我们这两种性格的人会是好朋友，这都无所谓了，我觉得没有哪两个人会那么性格相似，也没有谁规定必须是性格相似的人才可以做朋友。

经过我们多年的相处，她影响了我很多，改变了我很多，她是个性格比较强硬的女性，属于大女人，什么事情都要自己开心才行，所以她总是努力试图让周围的一切来适应她，而且她很有执行力，想得到什么就会为之竭尽全力，不达目的轻易不会罢休，她性格也大大咧咧，对小事情很少计较，每当我需要她的时候，不管她在做什么，她都会放下手里的一切来帮我，或者来找我。

她买东西向来都聚堆儿买（她的家境也不是比我好多少，她就是这个脾气，今朝有酒今朝醉），我每次都买不过她。对待朋友，她也不计较钱财，不像一般女人爱打小算盘，但是她比较现实，任何事都喜欢拿钱财来衡量，对成功人士很倾斜，她也一直想做个女强人，她现在确实要比我成功，不管是主观的还是客观的，但是我很为她高兴，我希望她好，我们俩很少会因为女人之间的那种小事发生矛盾，我们在一起基本不吵架，都是客观评价对方，另一方也会虚心接受，我们彼此鼓励，彼此扶持，我为有这样一个朋友而感到高兴。

我们俩的性格差异和价值观的差异我都可以接受，但是对待情感问题，我俩的观点也差很多，这就是我今天给您写信的原因。

我和她都结婚生子了，但是她经常对我说的就是她“博爱”，她基本没有同性朋友除了我，而我正相反，基本没有异性朋友。她有很多要好的男人，她说生活中不能只有老公，那样太无聊，要调剂，但是不要当真，还是要以老公为主，别人都可以随时随地扔掉，唯独老公会和她过一生，她也是这么做的，她的那些男朋友们也都和她观点一致，所以都相安无事。

但是几年前，她认识一个男人，这个男人跟她老公和之前她认识的男人都不太一样，属于比较顾家，比较纯情一点的（跟她之前的比），对她挺认真的（她是这么说的），他俩就像谈恋爱一样，很疯狂，她也一改之前的作风，一心一意的跟这个男人好，一度她甚至动了离婚的念头，是我强烈反对，她才消停，而且那个男人也曾经跟我说过，他绝对不会离婚的，他想得还是很清楚的，他俩就这么分分合合，吵吵闹闹的好几年了，中间分手过1年吧，后来又好上了，她说这个人已经是他的一个亲人、一种习惯了，她甚至说，这人是她将来的后老伴（如果她老公先他而去的话），而且那个人给他买了什么，或者给她什么，她都会告诉我，她很现实的，还和我说这个人还不够有钱，但是因为她爱他，她就凑合了，可我觉得她还是挺在乎的，那个男人如果定期不给她买点什么，她也很有意见的，她不承认，但我觉得是。

我说得太多了，抱歉，这些话憋在我心里好久了，我没有跟任何人透露过她的事情，包括我老公，所以一写就刹不住了，您可以挑着看。

她的经历一直让我很困惑，她还总劝我也找个男朋友，她每次和那个人吃饭都带着我，一是她觉得想和我分享美食，二是如果她老公查岗，我可以替她挡一下（她老公很敏感，她胆子很大），她总是在嘲笑我没有男朋友，她说，你怎么就不能找一个，咱们四个玩呢？我开始一直很排斥她的观点，还跟她据理力争，但是这么多年过去了，我跟她的男人都很熟了，我也不知道他们之间到底是不是真感情，我现在只能跟她说，sorry，这是你的人生观，不是我的，我可以理解你，但是你也别改变我了，她劝了我很多年，但是一直我都没改变，她也放弃了，但是她还是反复强调，说现在整个社会像她这样的是多数，我这样的是少数，甚至绝种。

我一边在质疑她的话，一边也在担心她的生活，我也很困惑，难道在爱的世界里，真的是偷情最美吗？两个相爱的人之间真的可以容下好多人吗？只爱一个人是无聊？无能？是情商低？这个社会真的都是人人皆有婚外情吗？还有，我这个闺蜜也经常和我探讨这方面的问题，您觉得我该不该劝劝她呢（她比较固执）？连岳，请您告诉我，好吗？

阿席达卡

阿席达卡：

朋友有两类方式得到，一是传统的线下型，或一起长大，或一起读书工作，混的时间多了，某方面契合，变成了朋友，你能容忍两人之间的观点差异。另一种是现代的线上型，三观互相认同，然后才在现实世界见面，这类不容易容忍观点差异，一方的观念一变，往往就是友情的结束。

据我观察，还是传统的友情占主流。这意味着为了保有共同的成长经验，人们更宽容一点，只要不起强行改变朋友的霸道之心，友情容易维持。不仅是我，很多人都会认同你说的：观念不同、性格不同的人可以成为朋友。友情不像爱情那么严苛，取其一点就行了。

你厌恶婚外情，闺蜜是婚外情高手，而你们可以成为朋友，都试图影响对方，都不受影响（至少目前是这样的），求同存异，这就是友情。

我想不出你有何可担心、可愤怒的，各过各的日子，无论是不是主流，适合你的、你又愿意负责任，就行。

从你的描述来看，你倒是在闺蜜的婚外情中起到了不该有的欺骗作用，她以你为掩护私会情人，你还一次次配合，算是这个骗局的从犯，不该让自己陷于这种境地，倒是在这点应该坚决退出，不能继续当王婆。

不要对闺蜜的婚外情太在乎，更不必过于愤怒，只要你不再参与欺骗，她自担后果，不关你的事。你对她的事想得太多，不是关心她，可能是在羡慕她（或者嫉妒她），也许你想像她一样生活，只不过没那个胆量。

对他人的私事不关心，不介入，允许他人的道德观、生活方式和自己不同，这将消除世上90%以上的烦恼。

随她去吧，她是个那么有主见的人，你别在她身上浪费精力。

祝开心。
